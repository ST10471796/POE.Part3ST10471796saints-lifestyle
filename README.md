# Saints Lifestyle Apparel

A premium South African lifestyle brand offering high-quality hoodies, beanies, and jeans.

## Project Overview

This website showcases Saints Lifestyle Apparel products with a modern, responsive design. The site includes e-commerce functionality, contact forms, SEO optimization, and interactive elements.

## Features

- Responsive design for all devices
- Product catalog with add-to-cart functionality
- Contact and enquiry forms with validation
- Interactive elements (accordion, lightbox gallery, maps)
- SEO optimization
- Accessibility features
- Automatic file download for enquiries
- Interactive map location

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (ES6+)
- Leaflet.js for interactive maps
- Font Awesome icons
